How it works
FIH will automatically create all 4 Notion databases (Watchlist, Research Notes, Earnings Calendar, Daily Digest) inside a page you choose. You keep full ownership — the databases live in your workspace.
Before you start: create any page in Notion, open it, click ⋯ → Connections and add your integration. Then copy the page ID from the URL and paste it below. The page ID is the 32-character string at the end of the page URL.
Note: the Notion API does not support setting column order on creation. Your databases will have all the right columns but you may need to reorder them manually in Notion. If column order matters to you, use Manual setup instead.
How it works
Duplicate our Notion template into your own workspace. You get the exact database structure FIH expects — with all columns, properties, and relations already configured.
Steps: open the template → click Duplicate → open each database in your workspace → copy the ID from the URL (the 32-char string) → paste below. Also make sure each database is shared with your Notion integration.
